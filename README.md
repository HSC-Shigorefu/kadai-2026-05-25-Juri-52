# データクリーニング・前処理 課題

この課題では、汚いデータを読み込み、分析しやすい形に整えてCSVとして保存します。

各課題フォルダに移動して、Pythonファイルを実行してください。

## インストール

最初に必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

# task01: 500行の社員CSVをきれいにする

フォルダ: `task01`

使用ファイル:

- `employees_dirty.csv`
- `task01.py`

出力ファイル:

- `employees_clean.csv`

やること:

- 500行の社員データを読み込む
- 列名を整理し、日本語の列名に統一する
- 社員ID、氏名、部署、勤務地などの文字列を整理する
- 重複している社員IDを処理する
- 部署、勤務地、雇用形態、ステータスの表記をそろえる
- 年齢、基本給、残業時間、評価点を数値として扱える形にする
- 入社日を日付として扱える形にする
- 不自然な値を欠損値として扱う
- 欠損値を列ごとに判断して補完する
- 必要な列だけを残し、見やすい順番に並べる
- きれいにしたデータをCSVとして保存する

# task02: 1つのSQLiteデータベースをきれいにしてCSV保存する

フォルダ: `task02`

使用ファイル:

- `company.db`
- `task02.py`

出力ファイル:

- `company_payroll_clean.csv`

やること:

- 1つのSQLiteデータベースに接続する
- `employees`、`payroll`、`attendance` の3つの表をSQLクエリーで取り出す
- それぞれの表の文字列、数値、日付を整理する
- 同じ社員IDが複数ある場合の扱いを決める
- CSVとして保存する

# task03: 3個のCSVを1つにまとめてきれいにする

フォルダ: `task03`

使用ファイル:

- `daily_reports/` フォルダ内のCSVファイル3個
- `task03.py`

出力ファイル:

- `daily_reports_clean.csv`

やること:

- 3個のCSVファイルを読み込む
- すべてのデータを1つの表にまとめる
- 社員ID、作業日、部署、プロジェクト、ステータスの表記を整理する
- 作業時間とコストを数値として扱える形にする
- 不自然な値を欠損値として扱う
- 欠損値を列ごとに判断して補完する
- 必要な列だけを残し、CSVとして保存する
